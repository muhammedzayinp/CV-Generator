# CV-Generator

This project generates clean, professional CVs using HTML templates. It's simple, lightweight, and helpful for quickly producing a resume you can customize and share.

## Description

This will be helpful — CV-Generator provides HTML templates and instructions so you can create and edit a CV in your browser or by editing the HTML file directly.

## Features

- Ready-to-use HTML CV template(s)
- Easy customization: edit text, colors, sections
- Print-friendly layout for PDF export

## Getting Started

1. Clone the repository:

   git clone https://github.com/muhammedzayinp/CV-Generator.git

2. Open the HTML file(s) in a browser:

   - Open `index.html` (or the relevant template file) in your web browser to view and edit your CV.

3. Edit the content directly in the HTML file or use the browser's developer tools to tweak styles.

## Usage

- Update your personal details, work experience, education, and skills in the template.
- Use your browser's Print dialog (Ctrl/Cmd+P) and select "Save as PDF" to export a PDF version of your CV.

## Customization

- Modify CSS in the included stylesheet(s) to adjust fonts, colors, spacing, and layout.
- Add or remove sections as needed.
